## Install:
* Uses no dependencies

**MacOS/Linux**
```bash
curl -sSL https://git.io/vQcAR | sh
```
```bash
=> geo -w

=> 12.23.56.78
```

**Tested in Bash, Zsh, Fish, Sh, and Csh**
